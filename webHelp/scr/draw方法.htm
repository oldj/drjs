方法: draw()

生成一个新画布对象（canvas）后，可以调用其draw方法在画布上绘制相应图形，方法如下：

canvas.draw(drawType, parameter[, left[, top[, color[, fill[, stroke]]]]]);

 

参数：

drawType

必填，表示绘图的类型，值可以为字符串、对象或数组。
如果值为字符串，则表示根据内置的绘图函数绘画，内置的绘图函数包括预置绘图函数和自定义绘图函数，例如"line"表示画直线的函数。如果输入的字符串在预置绘图函数中找不到，将抛出错误，错误消息为"drawType Error!"。

如果值为对象，表示传入一个方程，draw方法将根据这个方程绘图。方程的格式为：
{f: function(x) { ... return y}, interval: [x0, x1]}
相当于方程：
y = f(x), x∈[x0, x1]

如果值为数组，表示传入一个方程组，或者表示传入一个点阵。
方程组中每个方程的定义见上面方程的格式，点阵的格式如下：
[[x0, y0, width0, height0, color0], [x1, y1, width1, height1, color1], ..., [xn, yn, widthn, heightn, colorn]]
其中参数color可选，如果省略，将使用默认的颜色。

parameter

必填，表示给绘图方法传入的参数，具体格式因绘图方法的不同而不同。

left

选填，数值，表示图像距离画布左边的距离。默认为0。

top

选填，数值，表示图像距离画布顶部的距离。默认为0。

color

选填，表示图像的默认颜色，与CSS定义颜色的方式相同。如未指定，将依次从预置的颜色中读取，如预置的颜色已被读完将随机生成一个新的颜色。以下都是合法的颜色：
"red", "#ff0000", "#f00"

fill

选填，布尔型变量，表示图像是否填充。默认为不填充。
需要注意的是，并非所有形状都能填充成功，比如一个正弦函数的图形就无法填充。一般来说，封闭的形状（比如多边形、扇形等）可以填充。

stroke

选填，数值，表示画笔的粗细。默认为1。当参数fill设为true时，此项无效。

 

返回值：

一个形状（Shape）对象。
