方法: addFunc()

添加一个绘图方法。

drjs.addFunc(name, func);

 

参数：

name

字符串，必填。
表示添加的绘图方法的名字。

func

函数，必填。
表示该绘图方法的函数。

 

返回值：

undefined

 

说明：

添加一个绘图方法后，就可以在画布类的draw方法中直接使用，例如：

drjs.addFunc("myshape", function (canvas, p) { ... });
canvas.draw("myshape", parameters);

 

你还可以参考预置的绘图函数，使用addFunc()方法添加的函数与预置绘图函数具备同等地位。
