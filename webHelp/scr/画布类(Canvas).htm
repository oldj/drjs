画布类(Canvas)

DrJs的绘图都是在画布（Canvas）对象上进行的，生成一个画布对象可以说是使用DrJs绘图的第一步。

用法：

var canvas = new drjs.Canvas([parameter[, parent]]);

 

参数：

parameter:

可选，一个类似于CSS的对象。
各项值都是可选：
width: 画布宽度，number类型，单位为像素，默认值480
height: 画布高度，number类型，单位为像素，默认值360
backgroundColor: 画布背景色，默认为"#fff"
backgroundImage: 画布背景图片，默认为 "none"
borderWidth: 画布边框宽度，默认为0
borderStyle: 画布边框样式，默认为 "solid"
borderColor: 画布边框颜色，默认为 "#000"

parent:

可选，字符串或HTML文档对象
表示画布将要添加到的元素
如为字符串，则DrJs将查找id为parent的元素作为父元素
如未指定则为body元素

 

返回值：

一个Canvas对象。

 

示例：

var canvas = new drjs.Canvas({
  width: 480,
  height: 360,
  borderWidth: 1
}, "test");
